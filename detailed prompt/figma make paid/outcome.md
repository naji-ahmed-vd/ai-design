## Technologies Used

1. React
2. React Router
3. Tailwind
4. Recharts

## Pros

1. UI was extremely clean and had a feel to it.
2. Components like Header, Sidebar were created including a Layout.
3. UI was responsive except for the header & menu toggle button on mobile screen was overlapping it.
4. Created reusable components like Card on dashboard to keep UI consistent. 
5. Created some extra routes with text content in it.

## Cons

1. Extremly slow in generating the design and components, had to refresh page sometimes to make the changes reflect.
2. Sonnet 4.6 version on Figma seemed super slow and was unable to fix the basic issue.

## Outcome

The template that was generated was well polished in a single prompt, it had proper contrast of colors, created reuseable components and implemneted libraries for situation like charts and dropdowns.
