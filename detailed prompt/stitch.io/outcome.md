## Technologies Used

1. Tailwind CDN

## Pros

1. It didn't take a lot of time in creating the designs. 
2. It created both light and dark mode variants of the template.

## Cons

1. It didn't make the templates be mobile responsive.
2. Instead of implementing Chart library it used SVG vectors.
3. Instead of using reusable components it used HTML templates with Tailwind.
4. Notification icon button didnt have proper hover effect as it wasn't a complete circle.
5. In dark mode it didn't used proper color contrast as per WGAG guidelines.

## Outcome

The generated UI had a feel of being an example of a reference on how a design can be, it had some color contrast issues which were againat WGAG guidelines as per dev tools, the iterations can be improved with multiple prompts and selecting specific things to address them better.
