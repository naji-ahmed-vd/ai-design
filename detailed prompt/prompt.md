We are going to create a dashboard for a project named "Hospital". 

1. The dashboard will have a header with the project name, a search bar in middle and then a bell icon for notifications and afterwards we will show avatar of logged in user and a caret down icon to show the dropdown menu.
2. There should be a sidebar with the following sections:
    - Dashboard
    - Projects
    - Tasks
    - Reports
    - Settings
    - Logout
3. The sidebar should be fixed and should have a width of 200px.
4. The sidebar should be responsive and should have a toggle button to show/hide the sidebar.
5. In start of sidebar there should be a button named "Register patient" with plus icon.
6. After items in sidebar there should be a Settings button with gear icon.
7. The sidebar content should first contain four cards with the following content:
    - Total patients
    - Total appointments
    - Total prescriptions
    - Total invoices
8. There should be a card taking 70% of space and it should contain a chart with title "Outpaitents vs Inpatients Trends" along with a dropdown to select the time period.
9. The remaining 30% of space should contain another card with title "Patients by Gender" and a doughnut chart to show the distribution of patients by gender.
10. The last row should have a card with chart in it and show time based trends of patients admitted and discharged, another card with patients by division and a card with charts with patients in this month.
